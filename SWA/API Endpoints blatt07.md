
# Modul: Kundenverwaltung

| Ressource   | http-methode                                                | Request:<br>Parameter | Request:<br>Body    | Response:<br>Statuscode                   | Response:<br>Body                                                                                                                                      |
| ----------- | ----------------------------------------------------------- | --------------------- | ------------------- | ----------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| /kunden     | GET<br><br>Ziel: <br>Abfragen sämtlicher Kunden             | --                    | --                  | 200 OK<br>----<br>204 No Content          | Json-Array<br>sämtlicher Kunden<br>----<br>{"info": {<br>"message":<br>"(#1001)<br>Aktuell<br>können<br>keine<br>Kunden<br>übergeben<br>werden",}<br>} |
| /kunde/{id} | GET<br><br>Ziel:<br>Abfragen eines einzelnen Kunden<br><br> | id                    | --                  | 200 OK<br>----<br>404 Not Found           | Json des Kunden<br>----<br>kein Kunde gefunden                                                                                                         |
| /kunde      | POST                                                        | --                    | json mit attributen | 201 CREATED<br>----<br>400 BAD REQUEST    | Json des Kunden<br>----<br>kunde konnte nicht erstellt werden                                                                                          |
| /kunde/{id} | PATCH                                                       | id                    | json mit attributen | 204 NO CONTENT<br>----<br>400 BAD REQUEST | --<br>----<br>Kunde konnte nicht verändert werden                                                                                                      |
| /kunde/{id} | DELETE                                                      | id                    |                     | 200 OK<br>----<br>400 BAD REQUEST         | --<br>----<br>Kunde konnte nicht gelöscht werden                                                                                                       |
### Überlegung:
- Man zusätzlich kann ein Get/Post/Patch z.B. nur für die Adresse erstellen.
